Valorant is a First Person shooter Game created and launched by Riot games in 2020. It's a shooter game with agents integrated in its gameplay with different utility kits giving way for the user's experience not purely based on gunplay. There are 20 agents in the Valorant Protocol each having their own roles mainly Duelist, Initiator, Controller and Sentinel.
